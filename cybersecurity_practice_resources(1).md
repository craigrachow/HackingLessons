# 🛡️ Cybersecurity Practice Resources

A curated list of excellent online platforms and self-hosted labs for learning penetration testing, defensive security, CTFs and cybersecurity.

---

# 🌐 Online Practice Platforms

| Resource | Description | Account Required |
|---|---|---|
| [Hack The Box Academy](https://academy.hackthebox.com/) | Structured learning paths with hands-on labs covering Linux, Windows, Active Directory, web, cloud and more. | ✅ Yes |
| [Hack The Box Labs](https://app.hackthebox.com/) | Realistic vulnerable machines and challenges ranging from beginner to advanced. | ✅ Yes |
| [TryHackMe](https://tryhackme.com/) | Guided learning rooms and practical labs ideal for beginners through advanced learners. | ✅ Yes |
| [OverTheWire](https://overthewire.org/wargames/) | Classic wargames including Bandit that teach Linux, networking and security fundamentals. | ❌ No (SSH access) |
| [PortSwigger Web Security Academy](https://portswigger.net/web-security) | Free interactive labs covering the OWASP Top 10 and Burp Suite. | Optional |
| [Root Me](https://www.root-me.org/) | Hundreds of challenges covering web, crypto, reverse engineering, forensics and exploitation. | ✅ Yes |
| [PicoCTF](https://picoctf.org/) | Free CTF platform from Carnegie Mellon aimed at beginners. | ✅ Yes |
| [PentesterLab](https://pentesterlab.com/) | Practical web application security exercises and walkthroughs. | ✅ Yes |
| [Blue Team Labs Online](https://blueteamlabs.online/) | SOC, DFIR and blue team focused investigations and labs. | ✅ Yes |
| [VulnHub](https://www.vulnhub.com/) | Downloadable intentionally vulnerable virtual machines for offline practice. | ❌ No |
| [CyberDefenders](https://cyberdefenders.org/) | Blue team and digital forensics challenges using realistic datasets. | ✅ Yes |
| [RangeForce Community Edition](https://www.rangeforce.com/community-edition) | Interactive blue team training modules. | ✅ Yes |

# 💻 Self-Hosted Practice Labs

| Resource | Description | Account Required |
|---|---|---|
| [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) | Modern intentionally vulnerable web application covering dozens of common vulnerabilities. | ❌ No |
| [DVWA](https://github.com/digininja/DVWA) | Damn Vulnerable Web Application for learning web exploitation techniques. | ❌ No |
| [OWASP WebGoat](https://owasp.org/www-project-webgoat/) | Interactive lessons teaching common web application vulnerabilities. | ❌ No |
| [bWAPP](http://www.itsecgames.com/) | Vulnerable PHP application containing over 100 security exercises. | ❌ No |
| [Metasploitable 2](https://sourceforge.net/projects/metasploitable/) | Vulnerable Linux VM for exploitation and enumeration practice. | ❌ No |
| [Metasploitable 3](https://github.com/rapid7/metasploitable3) | Modern vulnerable lab built using Vagrant and Packer. | ❌ No |
| [Mutillidae II](https://github.com/webpwnized/mutillidae) | OWASP Top 10 practice application with multiple difficulty levels. | ❌ No |
| [OWASP Security Shepherd](https://owasp.org/www-project-security-shepherd/) | Self-hosted training platform with progressive web security exercises. | ❌ No |
| [DetectionLab](https://github.com/clong/DetectionLab) | Complete Windows enterprise lab for Active Directory, logging and detection engineering. | ❌ No |
| [BadBlood](https://github.com/davidprowe/BadBlood) | Automatically builds a populated Active Directory environment for lab practice. | ❌ No |
| [GOAD](https://github.com/Orange-Cyberdefense/GOAD) | Game of Active Directory - vulnerable enterprise AD environment for red team training. | ❌ No |
| [VulnLab Build Guides](https://vulnlab.com/) | Additional self-hosted lab ideas and vulnerable environments. | Some resources |
